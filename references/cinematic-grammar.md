## 焦段逻辑

| 焦段 | 适用场景 |
|:---|:---|
| 广角（<24mm） | 主观视角、强烈空间畸变、动作突兀感、环境压迫、第一人称代入 |
| 标准（35-50mm） | 客观平视、释怀、和解、带有呼吸感的人文视角 |
| 长焦（>85mm） | 空间压缩、中产阶级牢笼感、偷窥感、极度孤立的心理特写 |

## 景深逻辑

| 景深 | 标注 | 适用场景 |
|:---|:---|:---|
| 浅景深 | 浅 | 孤立人物、主观聚焦、情绪特写、排除环境干扰 |
| 中景深 | 中 | 常规叙事，人物与环境保持自然关系 |
| 深景深 | 深 | 环境>人物、客观全貌、强调空间关系、群像调度 |

## 运镜逻辑

| 运镜方式 | 适用场景 |
|:---|:---|
| 固定机位 | 客观凝视、压抑、秩序感、让表演说话 |
| 手持 | 不安、纪实感、混乱、主观紧张 |
| 斯坦尼康 | 流畅跟随、长镜头调度、空间连续性 |
| Dolly 推拉 | 心理揭示（推）/心理疏离（拉）、空间关系变化 |
| 摇臂升降 | 宏大感、脱离感、宿命"上帝视角" |
| 摇摄（Pan/Tilt） | 信息揭示、视线引导、空间关系展现 |
| 跟拍（Tracking） | 跟随主体移动，保持空间关系。分三种：**前跟**（镜头在主体前方后退）/**后跟**（镜头在主体后方跟随）/**侧跟**（平行移动，如横移跟拍）。Dolly 后退+跟随=拉跟 |

## 角度与构图逻辑

### 相机机位（Camera Position）

机位是分镜表中「机位」列的五种取值。底层逻辑源自好莱坞三角机位系统——只要确立了人物之间的「动作轴线」，所有机位都在轴线一侧的半圆形工作区间内派生。

**五种机位：**

| 机位 | 定义 | 子情况 | 观众位置 | 典型场景 |
|:---|:---|:---|:---|:---|
| **正打** | 相机在角色面前 | ① 前景不挂：干净单人/斜角单人/POV 主观<br>② 前景挂：过肩/过腰/过手/过腿 | 面对角色——我在看「他」 | 情绪反应镜、对白、破发点特写 |
| **反打** | 相机在角色身后或对面，与正打对称 | ① 前景不挂：单人背影/POV 所见<br>② 前景挂：越过角色看对面 | 在角色身后——我成了「他」；或在对面——我看着「他」的背影远去 | POV 主观镜、动作→反应的正反打交替、离场背影 |
| **第三人称** | 相机不绑定任何角色视角，客观旁观 | 涵盖双人主镜/单人主镜/侧拍/群像/俯拍 | 旁观——我不在故事里 | 多角色调度、侧拍跟随、主镜头覆盖 |
| **插入/切出** | 跳脱三角机位系统。①不在当前场景 3D 空间内（闪回、切出到另一空间），或②在当前空间内但不在动作→反应链中（轴线过渡的空镜、纯标点） | 闪回、另一空间的画面、过渡空镜 | 被引导——导演指给我看 | 时空跳转、情绪标点、轴线过渡救急 |
| **空间环境** | 纯空间交代。全景或远景，角色在画面中占比 <20%，功能是建立或交代空间关系 | 开场建立/转折建立/闭合建立/闪回空间建立 | 被安置——导演告诉我这是哪 | 全景交代空间关系、质感建立（光/色/颗粒） |

> **注意区分「机位」和「景别」：** 机位是相机和角色的空间关系（我在他面前/身后/旁边），景别是取景范围（我离他多远）。正打+特写 = 我在他面前很近的地方盯着他的脸。第三人称+全景 = 我远远地、不参与地看着整个空间。插入+极特写 = 我不关心空间关系，我只想看那个细节。

> **「插入/切出」和「空间环境」的机位是无意义的：** 插入/切出跳脱三角系统，空间环境是纯地理——都不需要标注正反打子情况。

> **正反打的锚点规则：** 正打和反打不是纯几何描述——它们的语义锚点是本场戏的**视点角色**（在模块1分析报告的「视点角色」字段声明）。
> - **正打 → 主语是视点角色。** 观众看视点角色的表情和反应。包括：视点角色的脸（前景不挂·干净单人）、视点角色的身体动作（前景不挂·斜角单人）、越过他人肩头看视点角色（前景挂·过肩/过腰）。
> - **反打 → 宾语是视点角色的视线对象。** 观众看视点角色在看的东西。包括：视点角色所见的画面（前景不挂·POV 所见）、从视点角色身后方向拍摄他所面对的人/物/环境（前景不挂·单人背影，前景挂·越过角色看对面）。
> - **都不是 → 按以下优先级：** ④ 在当前空间内 + 在动作→反应链中 → 第三人称。⑤ 在当前空间内 + 不在动作→反应链中（纯标点空镜）→ 插入/切出。⑥ 不在当前空间内（闪回等）→ 插入/切出。⑦ 全景或远景 + 角色占比 <20% + 功能为交代空间关系 → 空间环境。

## 音乐设计逻辑

音乐不是后期才想的事。分镜阶段必须定下每场戏的音乐策略。

**音乐列填写格式：** `[情绪方向] | [配器/风格] | [速度] | [入出点]`

| 参数 | 说明 | 示例 |
|:----|:----|:----|
| 情绪方向 | 用 2-3 个词描述该镜的音乐情绪走向 | `压→放`、`悬→紧→炸`、`静→悲` |
| 配器/风格 | 主要乐器和风格 | `低音弦乐`、`钢琴独奏`、`电子pad+心跳音效` |
| 速度/密度 | 节拍感或疏密 | `慢弓 50BPM`、`碎拍渐快`、`静` |
| 入出点 | 在哪镜开始/结束 | `入：本镜`、`出：镜12止`、`—`（故意无音乐） |

> 深入的配器择选、全片音乐弧线设计、Motif 设计等，见 `references/music-design.md`。

### 音乐设计总思路

1. **音乐情绪弧线 = 价值移动链的音乐翻译。** 节奏曲线中"情绪温度 8/10"的段落，音乐应该比"情绪温度 3/10"的段落更密、更响、更快。反过来也可以——情绪压抑时让音乐提前推进（预示），或情绪爆发时让音乐撤空（留白让表演本身说话）。
2. **为重复出现的物件/人物设计音乐主题（Motif）。** 通过线追踪中的物件（币、手环、心率）每次出现时，用同一个旋律变奏——观众未必注意到，但潜意识会连上。
3. **"没音乐"是一个音乐决策。** 不是"忘了加"——在分镜音乐列里标注 `—` 或 `（无声）`，明确告诉后期这里不需要音乐。
4. **音画对位：** 欢乐的画面配悲伤的音乐比配欢乐的音乐更有力量。音乐不一定要跟着画面情绪走，有时反着来效果更强。

## 光影逻辑（粗略定调，细节留待概念设计阶段）

## 节奏控制

物理时长精确预估，参考范围：

| 镜头类型 | 典型时长 |
|:---|:---|
| 快速闪切 / 生理特写 | 0.5s - 1.5s |
| 标准叙事镜头 | 2s - 4s |
| 沉重的反应镜头 | > 4s |
| 长镜头 / 余韵 | > 6s |

## 戏点镜头景别铁律

核心戏点的镜头组必须包含至少一个 ECU（极特写）或 CU（特写），用于捕捉情绪转折时的生理微反应。

典型拆解模式：
- **镜 A：** 中景/全景 — 建立空间，展示动作的初始状态
- **镜 B：** 极特写/大特写 — 破发点，情绪爆发的生理细节（手指颤抖、瞳孔收缩、嘴角抽动）
- **镜 C：** 反应镜头 — 他人/环境对破发点的回应，完成闭环

## 多角色轴线管理

- **双人场景：** 确定主要对话对（A-B）为基准轴线，全程维持
- **三人及以上：** 确定主要对话对（A-B）为基准轴线；第三人入画时通过视线方向或身体朝向暗示空间关系
- **轴线偏移：** 角色走位导致轴线改变时，必须插入中性过渡镜头（如空镜/全景/切出），再建立新轴线
- **空间坐标追踪：** 每个镜头必须明确当前角色的画面位置和朝向

## 空间状态追踪

每生成一个镜头前，先在内心声明当前空间状态（无需写入输出，但必须确保逻辑一致）：

```
当前空间状态：角色A(画面左, 面向右) / 角色B(画面右, 面向左)
→ 本镜头为角色A主观视角，面向右，不改变轴线
→ 下一镜头切入角色B反应，需维持角色B在画面右、面向左
```
