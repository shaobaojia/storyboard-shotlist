# 分镜音乐设计方法论

> 与 storyboard-shotlist SKILL.md 的「音乐设计逻辑」配合使用。提供更深入的原理和实操指引。

---

## 一、核心原理：音乐是情绪价值移动的声音翻译

参见 screenwriter skill 的 `references/scene-audit.md`「价值移动链」。全片的价值移动链是音乐的蓝图：

```
控制 → 羞辱 → 恐惧 → 虚假恢复 → 瘫倒 → 召唤
```

**对应音乐弧线：**

| 价值段 | 音乐策略 |
|:------|:---------|
| 控制（平静日常） | 无音乐 或 环境音（空调嗡鸣、键盘声） |
| 羞辱 | 低频轰鸣 / 心脏搏动声 / 开始有低音弦乐 |
| 恐惧 | 不协和音程（dissonance），碎弓，拍频加速 |
| 虚假恢复 | 轻柔钢琴单音——但旋律有隐藏的不安感（减和弦） |
| 瘫倒 | 音乐骤停（sforzando 切断），只剩耳鸣/环境声 |
| 召唤（童年闪回） | 八音盒/铃声——完全不同音色的侵入，标志另一个时空 |

## 二、音乐列的四个维度详解

### 2.1 情绪方向

用箭头链描述音乐在这一镜中的情绪走向。箭头表示在该镜内发生的变化。

| 写法 | 含义 |
|:----|:-----|
| `压` | 一个方向贯穿全镜 |
| `压→放` | 从压抑到释放（该镜内有转折） |
| `悬→紧→炸` | 悬疑→紧张→爆发（多段递进） |
| `静→悲` | 从安静的铺底转为悲伤 |
| `放→收` | 从爆发突然收住（用于反应镜） |

### 2.2 配器/风格

一个镜头的配器不应超过 2-3 种乐器（除非是高潮段落）。配器选择直接决定片子的气质：

| 配器 | 情绪/场景指向 |
|:----|:------------|
| 钢琴独奏 | 孤独、回忆、纯粹情感 |
| 低音弦乐群 | 压抑、威胁、悲怆 |
| 小提琴独奏（solo violin） | 脆弱、疼痛、个人的 |
| 八音盒/钢片琴/铃声 | 童年、梦境、另一个世界 |
| 电子 pad + 氛围 drone | 悬疑、抽离、现代感 |
| 大提琴 | 沉重、失去、衰老 |
| 心跳音效（合成或拟音） | 生理紧张、恐惧、濒死 |
| 单簧管/双簧管 | 悲伤的田园感、怀旧 |
| 竖琴 | 梦境、回忆过渡、轻柔的时光感 |
| 定音鼓/大鼓 | 威胁逼近、命运敲门 |
| 静（无配器） | 最有力的选择——让其他感官接管 |

### 2.3 速度/密度

不是精确的 BPM 数字，而是音乐的疏密感。

| 标记 | 场景示例 |
|:----|:---------|
| `慢弓 50BPM` | 沉重步态、压抑氛围 |
| `中板 80BPM` | 正常叙事、行走、日常 |
| `快板 120BPM` | 追逐、兴奋、爆发 |
| `碎拍渐快` | 紧张度递进、危险逼近 |
| `自由节奏（rubato）` | 不确定、漂浮、梦 |
| `极慢` | 濒死、时间的拉长感 |
| `静` | 无节拍感，让画面节奏主导 |

### 2.4 入出点

| 写法 | 含义 |
|:----|:------|
| `入：本镜` | 音乐从本镜开始 |
| `入：镜05` | 音乐从之前某镜延续下来（用于空置的过渡列） |
| `出：镜12止` | 音乐在本镜末尾停止 |
| `—` | 明确无音乐 |

## 三、全片音乐情绪弧线图

分镜全部做完后，从每场的音乐列汇总出一张全片音乐弧线图：

```markdown
## 全片音乐情绪弧线

| 场次 | 首镜 | 末镜 | 情绪弧线 | 配器主轴 | 最高密度镜 |
|:----|:----|:----|:---------|:---------|:----------|
| s010 | 01 | 18 | 压→召 | 环境音→弦乐→八音盒 | 08（童年闪回） |
| s020 | 19 | 30 | 悬→愧 | 弦乐→钢琴 | — |
| ... |
```

这张表就是整个片子的「音乐设计需求文档」，可以直接交给作曲家或喂给 Suno/AIVA。

## 四、Motif（主题旋律）设计

通过线追踪（scene-audit.md）中发现的重复物件/数字/空间，为每一个设计一个短小的音乐母题：

| 通过线 | 建议 Motif | 每次出现时的变奏 |
|:------|:-----------|:----------------|
| 那枚币 | 清脆的高音单音（钢片琴或三角铁） | s010 第一次滚来→单音。s080 弹向镜头→同样的单音但加了钢琴和弦（完整） |
| 心率手环 | 低频脉冲音（与心跳同频） | s010 110BPM 正常→s050 150BPM 急促失稳→s060 110BPM 但掺了杂音 |
| 摩托车机台 | 电吉他泛音或 8-bit 音色 | 童年闪回用干净 8-bit，现实用失真效果 |
| 跳舞机上的年轻人 | 远处传来的弱拍电子鼓 | 背景音，不被主角注意 |

Motif 不需要长，2-4 个音就够了。观众不需要认出"啊这是那个主题"——潜意识里有感觉就行。

## 五、留白设计（何时不该有音乐）

比"哪里有音乐"更重要的是"哪里没有"：

| 适合留白的场景 | 理由 |
|:-------------|:-----|
| 开场平凡世界（s010 前半） | 让观众先进入真实感，音乐只在情绪转折时才介入 |
| 高潮前的死寂（s060 攥紧→松手） | 没有音乐时，呼吸声/心跳声/硬币滚动的才是最响的 |
| 结尾余韵（s080 最后 20 秒） | 让画面本身收尾，不在情绪上画蛇添足 |
| 幽默段落的中段（s030 闪腰） | 喜剧效果来自"意外"，音乐提前暗示了就不好笑了 |

## 六、将分镜音乐列翻译成 Suno / AIVA prompt

分镜音乐列填写好之后，汇总就是 AI 音乐工具的 prompt 素材：

```
分镜音乐列：压→放 | 低音弦乐渐起 | 慢弓 | 入：本镜
→ Suno prompt: Cinematic orchestral, slow cello and double bass, building tension, 
   instrumental, no vocals, 50 BPM, emotional, anticipatory

分镜音乐列：悲→念 | 钢琴独奏+八音盒 | rubato | 入：本镜
→ Suno prompt: Sad piano solo, music box melody in the distance, nostalgic, 
   rubato tempo, minimalist, instrumental, cinematic
```

注意事项：
- Suno 的 `Instrumental` 模式产生纯器乐，适合 BGM
- AIVA 更适合管弦乐/氛围/电影配乐风格，且支持 MIDI 导出导入 DAW 精修
- 一条情绪可能需要 5-10 次生成才能挑到满意的，迭代是正常的
